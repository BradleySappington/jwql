About the project

The James Webb Quicklook Application (JWQL) is a database-driven web application and software repository for use by the JWST instrument teams. Developed in Python, it exists within a private GitHub repository. The team actively strives to adhere to software engineering best practices, strictly following defined workflows and Python style guides.

Contact

If you have discovered an issue with the functionality of the JWST Quicklook project, we ask that you please submit an issue on our GitHub repository. Any other questions, comments, or concerns can be directed to jwql@stsci.edu.
